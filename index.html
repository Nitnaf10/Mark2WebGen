Comment utiliser Mark2WebGen

Mark2WebGen est un outil simple pour générer du code HTML à partir de Markdown et de CSS. Voici comment l'utiliser :

1
Écrire en Markdown : Dans la zone de texte "Écrivez votre Markdown ici...", saisissez votre texte en utilisant la syntaxe Markdown. Par exemple, utilisez des astérisques pour le texte en gras ou des dièses pour les titres.

2
Ajouter du CSS : Dans la zone de texte "Écrivez votre CSS ici...", vous pouvez ajouter vos styles CSS personnalisés pour styliser le HTML généré.

3
Voir le résultat : Le code HTML généré apparaîtra automatiquement dans la zone "Code HTML généré ici...". Vous pouvez également voir le rendu visuel dans la section de sortie.

4
Appliquer des styles : Utilisez les boutons sous "Styles" pour appliquer des styles prédéfinis aux éléments de type "blockquote". Vous pouvez également personnaliser les puces de vos listes en saisissant un symbole dans le champ prévu à cet effet.

Amusez-vous à créer et à styliser votre contenu web avec Mark2WebGen !
